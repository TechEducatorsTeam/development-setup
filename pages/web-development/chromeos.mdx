# ChromeOS

You'll need to have the latest ChromeOS edition. Which at time of writing is ChromeOS 134.

The easiest way to check if you system is up to date and get any updates is to follow the excellent instructions by Google which you can find [here](https://support.google.com/chromebook/answer/177889?hl=en-GB#:~:text=Settings%20.,operating%20system%20your%20Chromebook%20uses)
