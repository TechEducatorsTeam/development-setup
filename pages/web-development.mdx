# Web Development Setup

Let's get you setup and ready to build websites and web applications on your
Laptop or PC.

The minimum techinal specifications for a student computer:

- Processor: Intel Core i5 (or equivalent)
- 16 GB RAM
- Hard disk space: at least 20 GB
- A connection to the internet
- Operating system: Window 10/11, Linux, MacOS Ventura

You will also need to have:

- Keyboard and mouse/trackpad
- Webcam
- Microphone

We'll install:

- [Git](https://git-scm.com/) VCS.
- [Node.js v18](https://nodejs.org/) via [nvm](https://github.com/nvm-sh/nvm).
- [Visual Studio Code](https://code.visualstudio.com/) aka VSCode.
