# Web Development Setup

Let's get you setup and ready to build websites and web applications on your
Laptop or PC.

The minimum techinal specifications for a student computer:

- Processor: Intel Core i5 or AMD Ryzen 5
- 16 GB RAM
- Storage space: at least 20 GB (we reccommend a SSD but a HDD is fine)
- A good connection to the internet
- Operating system: Window 10/11, Linux, MacOS Ventura (we offer support for ChromeOS however it is not recommended)

You will also need to have:

- Keyboard and mouse/trackpad
- Webcam
- Microphone

We'll install:

- Git Version Control System (VCS).
- Node.js v18 & Node Version Manager (NVM).
- Visual Studio Code (VSCode).
