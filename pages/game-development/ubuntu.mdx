# Installs Unity Hub

<Steps>
  ### Step 1: Create a Unity ID

  You can register for Unity [here](https://id.unity.com/en/conversations/adc07ba5-7f26-4ad9-a8e0-069d3cd78fec007f),
  if you already have an account, you skip this.

  [Image of Create Unity Account]

  ### Step 2: Add the Unity aptitude repositories

  - To add the public signing key, run the following command:
  
    ```sh copy
    wget -qO - https://hub.unity3d.com/linux/keys/public | gpg --dearmor | sudo tee/usr/share/keyrings/Unity_Technologies_ApS.gpg > /dev/null
    ```
  - To add the Unity Hub repository, you need an entry in /etc/apt/sources.list.d.
    Run the following command to add the Unity Hub repository:
  
    ```sh copy
    sudo sh -c 'echo "deb [signed-by=/usr/share/keyrings/Unity_Technologies_ApS.gpg] https://hub.unity3d.com/linux/repos/deb stable main" > /etc/apt/sources.list.d/unityhub.list'
    ```
  - Update Aptitude's package cache
  
    ```sh copy
    sudo apt upgrade -y
    ```

  ### Step 2: Install the Unity Editor & Modules

  1. Install Ubuntu Hub from Aptitude

    ```sh copy
    sudo apt install -y unityhub
    ```
  2. Sign in to your Unity ID by selecting the drop down at the top left corner of the screen.
    [Image of Sign-in]
  3. Select **Unity 2023.3.5f1 LTS**
    [Image of Ubuntu version selector]
    <Callout emoji="ðŸ’¡">
      Add Microsoft Visual Studio code by selecting the checkbox.
    </Callout>
  4. Click on the Install button at the right bottom corner of your screen.
    [Image of Install screen]
</Steps>
